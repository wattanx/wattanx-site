はじめて自分のブログでふりかえりを書きました。

1月

- Nuxt Bridge リリース 🎉
  - https://github.com/nuxt/bridge/releases/tag/v3.0.0
- 雪で出社失敗
- Nuxt Bridge を使い始めた当初から困ってた問題を解消した
  - https://github.com/nuxt/bridge/pull/1070
- [Tanstack Table](https://tanstack.com/table/latest) を業務で使い始める
  - Vue 2 に対応してなかったので、Vue 2 用の wrapper を作った

2月

- ひたすら業務でいい感じの Table コンポーネントを作る
- `Nuxt Server Components`を完全に理解する

3月

- v-tokyo で`@nuxt/test-utils`の話をする
  - [Nuxt Test Utils で始める効率的な Nuxt アプリケーションのテスト](https://speakerdeck.com/wattanx/getting-started-with-nuxt-test-utils-for-efficient-testing-nuxt-applications)
- `@nuxt/test-utils`を完全に理解したので、nuxt 2 の runtime 環境で動くライブラリを作ろうとする
  - 忙しくなってきたので諦める
- Nuxt Bridge v3.1 Release 🎉

4月

- `Nuxt Server Components`における`Teleport`の使い方を完全に理解した
- `Tanstack Table`を使った機能がリリースされた
- CSS Animation について勉強し始めた

5月

- Vercel の [AI SDK](https://sdk.vercel.ai/) を使って、サイトマップを自動で生成するツールを作ったりした
- フロントエンドカンファレンス北海度の CFP が落選
- [uilabs.wattanx.dev](https://uilabs.wattanx.dev/) を作った
- Nuxt Bridge v3.2 Release 🎉

6月

- 新規事業のことを考えつつ、Nuxt 3 移行をがっつりすすめる
  - この時期に Vite 化もした。
- Nuxt Bridge の release workflow を整えた
- 新規プロダクトを Next.js で作る決意をした
- Nuxt Bridge v3.3 Relase 🎉

7月

- Vue Fes の CFP を出した
- 28歳になった
- 新しいモニターを買った
- v-tokyo で Nuxt 4 への移行は大変じゃないという話をした
  - https://wattanx-v-tokyo-21.pages.dev/
- Nuxt 3 移行完了
- 社内のライブラリのドキュメントを`vitepress`に移行し始めた
- height の animation を試した
  - https://uilabs.wattanx.dev/animating-height/

8月

- [nuxt-bundle-analysis](https://github.com/wattanx/nuxt-bundle-analysis) の vite 対応完了。
- Vue Fes の CFP が採択された

9月

- `Nuxt Server Components`を再び完全に理解した
- `Nuxt Server Components`の example 集をつくった
  - https://nuxt-sc-example.pages.dev/
- CSS を書くのが楽しいと思えるようになった
- 「ゼルダの伝説 知恵のかりもの」 を買った

10月

- `@nuxt/test-utils`のコミット権限をもらった
- Nuxt Bridge の nightly release に [pkg.pr.new](https://github.com/stackblitz-labs/pkg.pr.new) を導入した
- 自分のブログの OG Image のデザインを変えた
- Vue Fes Japan 2024 に登壇した
  - [Deep dive into Nuxt Server Components](https://speakerdeck.com/wattanx/deep-dive-into-nuxt-server-components)
- Vue Fes のアフターイベントに登壇した
  - [Unlocking the potential of Nuxt Server Components](https://speakerdeck.com/wattanx/unlocking-the-potential-of-nuxt-server-components)
- Vue の`effectScope`を完全に理解した
- `nodenv`をやめて`mise`にした

11月

- Figma Plugin デビュー
  - Figma Variables を export したかった
- ドラクエⅢ を買った

12月

- 新規プロダクトのリリース
- Vuex -> Pinia に置き換える tool をつくった
  - https://wattanx-converter.vercel.app/pinia-converter/
- [Nuxt で Next.js の Private folders のような機能を実現する方法](https://zenn.dev/comm_vue_nuxt/articles/private-folders-for-nuxt) を書いた
